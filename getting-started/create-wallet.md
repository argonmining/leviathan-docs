# Create a wallet

For pioneers, the default path is the **Leviathan Chrome extension**. The hosted web wallet remains available if you need a browser-tab option.

## Recommended: Chrome extension

Follow [Install the Chrome extension](../wallets/chrome-extension.md): get the package from the **Pioneers Telegram** pin, **Load unpacked** in Chrome, create a wallet, and copy your **bech32** receive address.

Then:

1. [Get testnet funds](request-funds.md)
2. [Send tokens](send-tokens.md)

## Secondary: hosted web wallet

Full steps: [Hosted web wallet](../wallets/web-wallet.md).

Short version:

1. Open [leviathandev.neptune.io/wallet](https://leviathandev.neptune.io/wallet).
2. **Connect Wallet** → **Create** an account.
3. **Sync** after funding or receiving.
4. Use hex or bech32 where the hosted UI accepts either format.

## Which should I pick?

| Need | Use |
|------|-----|
| Primary pioneer experience | Chrome extension |
| No extension install possible right now | Hosted web wallet |
| Mobile app store install | Not offered in public pioneer docs (internal only) |
